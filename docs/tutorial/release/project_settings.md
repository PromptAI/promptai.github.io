项目设置位于左侧导航窗格中的项目工具菜单下。

## 概述
项目设置包括以下几项：
- 姓名
- 语言
- 描述
- 欢迎辞
- 倒退
- 图标

除了上面的配置之外，还有其他的配置项：
- 将常见问题解答和流程呈现为按钮的选项，
- 常见问题解答和每个流程的第一条消息。
